# Changelog

## v0.1.3
*   **UI**: 更新左下角版本顯示至 v0.1.3。
*   **Maintenance**: 專案版本號更新。
*   **Build**: 支援自動產出 Windows (`.exe`/`.zip`) 與 macOS (`.dmg`/`.zip`) 兩種平台的執行檔。

## v0.1.2 測試版
*   **DevOps**: 新增 GitHub Actions workflow，支援自動化建置 Release。

## v0.1.0 測試版
*   **核心功能**: 完整的 12 週計畫管理功能，包含 Dashbaord, Weekly Plan, Goals Strategy。
*   **本地資料庫**: 整合 SQLite 進行本地資料儲存。
*   **編輯器**: 整合 Tiptap 富文本編輯器與 CodeMirror 程式碼區塊支援。
*   **介面**: 採用現代化深色模式 (Dark Mode) 與響應式側邊欄設計。
